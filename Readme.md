## 🕹 ゲーム概要

**Horizon Survivor**は、プレイヤーが生き延びることを目指す3Dサバイバルゲームです。敵となるNPCから逃げながら、限られた資源で生存時間を延ばしてください。夜まで生き延びることができれば勝利です。

---

## 🎯 ゲームの目的

* 夜になるまで生き延びれば勝利です。
* プレイヤーには2つの体力システムがあります:
  * 通常の体力（時間と共に減少）
  * キック耐久（敵に蹴られると減少）
* どちらかが0になるとゲームオーバーになります。
* 「酸素缶」を拾うことで体力を回復できます。
* 「ツールキット」を拾うことでブーストが使用可能になります。

---

## 🎮 操作方法

* `WASD`：移動
* `Space`：ジャンプ
* `Shift`長押し：走る（スプリント）
* `Alt` + 移動：ブースト使用（ブースト残数がある場合）
* マウス移動：視点操作

---

## 🔧 推奨設定・注意事項

* 推奨解像度:2560x1440(フルスクリーン)
* 高負荷の環境では、グラフィック設定を「中」以下に下げることを推奨します。
* シーン内のモデルにはオクルージョンカリングおよびベイクされたライティングが適用されています。
